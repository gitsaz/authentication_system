এটি একটি Django Authentication System প্রজেক্ট, যেখানে ব্যবহারকারীদের জন্য নিরাপদ ও সম্পূর্ণ Authentication ব্যবস্থা তৈরি করা হয়েছে। এই প্রজেক্টে Custom User Model ব্যবহার করা হয়েছে এবং আধুনিক ওয়েব অ্যাপ্লিকেশনে প্রয়োজনীয় প্রায় সব Authentication ফিচার যুক্ত করা হয়েছে।

এখানে ব্যবহারকারী Registration করতে পারে, Login ও Logout করতে পারে, নিজের Password পরিবর্তন করতে পারে এবং Email এর মাধ্যমে Password Reset করতে পারে। Login ছাড়া কোনো ব্যবহারকারী Protected Page অ্যাক্সেস করতে পারবে না। এছাড়াও Permission ও Access Control ব্যবহার করে সিস্টেমটিকে আরও Secure করা হয়েছে।

এই প্রজেক্টটি Django এর Built-in Authentication System, Class-Based Views, Generic Views, Messages Framework এবং Bootstrap ব্যবহার করে তৈরি করা হয়েছে, যাতে কোড ক্লিন, রিইউজেবল এবং রিয়েল-ওয়ার্ল্ড প্রজেক্টের মতো হয়।
